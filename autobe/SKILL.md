# AutoBE

This skill integrates `autobe`, an AI backend builder that analyzes requirements in natural language and builds backend applications.

## Local Repository
The repository is fully cloned and available locally at:
`C:\Users\mohan\Downloads\PROJECT 1 ANTIGRAVITY\autobe`

## Features
- **Natural Language to Backend**: Describe requirements through a chat interface.
- **100% Buildable Code**: Uses AI-friendly compilers to guarantee type safety (TypeScript, Prisma).
- **Type-Safe Client SDK**: Auto-generates a client SDK alongside the backend.
- **Testing**: Includes e2e testing generation.
- **Local LLM Support**: Can use models like `qwen3-next-80b-a3b-instruct`.

## Start the application
1. Navigate to the repository: `cd "C:\Users\mohan\Downloads\PROJECT 1 ANTIGRAVITY\autobe"`
2. Install dependencies: `pnpm install`
3. Run the playground: `pnpm run playground`
4. Access the playground at `http://localhost:5713`
