# Cloudflare VibeSDK

This skill integrates the `cloudflare/vibesdk` repository. It provides an open-source AI platform to allow users to build React + TypeScript + Tailwind web applications using natural language.

## Local Repository
The repository is fully cloned and available locally at:
`C:\Users\mohan\Downloads\PROJECT 1 ANTIGRAVITY\vibesdk`

## Key Features
- **AI Code Generation**: Phase-wise development using LLMs via AI Gateway.
- **Live Previews**: Sandboxed app previews deployed dynamically.
- **Cloudflare Native Stack**: Builds on Workers, Durable Objects, D1 (SQLite), and R2.
- **Interactive Chat**: Natural language interface for generating full-stack apps.
- **Programmatic SDK**: Allows automated app generation through `@cf-vibesdk/sdk`.

## Getting Started
1. Navigate to the repository: `cd C:\Users\mohan\Downloads\PROJECT 1 ANTIGRAVITY\vibesdk`
2. Run `bun install` or `npm install`.
3. Run `bun run setup` to configure databases, credentials, and templates.
4. Launch locally using `bun run dev` (running on `http://localhost:5173`).
